# Airflow-migration


## Installation
Create an virtual environment

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.tx
```

## Install Airflow


```bash
 bash install_airflow.sh
```

## Usage
Airflow is available at ```http://localhost:8080/ ```
